\section*{Team Contributions}
\textbf{Ganesh} led the architecture definition, wrote the specification for the ISA, designed the memory map, and implemented the emulator entry-points and CLI tooling.\newline
\textbf{Tushar} focused on core CPU logic, implementing the ALU, status flag semantics, and the control unit that sequences fetch/compute/store cycles.\newline
\textbf{Jeevan} created the assembler, directive support, label resolution logic, and authored the example programs (Hello World, Fibonacci, Timer).\newline
\textbf{Sindhura} produced the documentation set (architecture, emulator, assembler, programs), the CPU schematic, the Fibonacci demo video, and compiled this report.
